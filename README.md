## Installation

1. Open **Nuvio** app
2. Go to **Settings → Local Scrapers**
3. Add this repository URL:
   ```
   https://raw.githubusercontent.com/NuvioPlugin/All-in-One-Nuvio/refs/heads/main/manifest.json)
   ```
4. Enable the scrapers you want to use
